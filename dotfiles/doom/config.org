*** Golden ratio
I like splitting my editor window so that I can see more than one file that I'm working on. Sometimes it's essential. For example, if I'm working on a Terraform module, and I want to keep an eye on several bits of relevant info.

I don't like the way that Emacs handles this. Window sizes are usually 'wrong', and I spend too much time resizing them. Golden Ratio might fix this for me. I'm starting out with letting GR handle things automatically, but a max width of 120 characters.

There are some buffers that should never be scaled. They are added to =golden-ratio-exclude-buffer-regexp=.

That being said, Zoom is set up below and this block is currently disabled. Zoom might do a better job. Golden Ratio also appears to be unmaintained at the moment.
#+begin_src emacs-lisp :tangle no
(golden-ration-mode 1)
(setq golden-ratio-auto-scale t)
(setq golden-ratio-max-width 120)
(add-to-list 'golden-ratio-exclude-buffer-regexp "^\\*[hH]elm.*")
(add-to-list 'golden-ratio-exclude-buffer-regexp "^\\*vterm.*")
(add-to-list 'golden-ratio-exclude-buffer-regexp "^\\*ediff.*")

#+end_src

*** Zoom
Automatically resize the active window.

If the frame size is >1280 then use 100 columns and 75% of the frame height. Otherwise, make it a 50%x50% split.
#+begin_src emacs-lisp :tangle yes
(zoom-mode)
(defun size-callback ()
  (cond ((> (frame-pixel-width) 1280) '(100 . 0.75))
        (t                            '(0.5 . 0.5))))

(custom-set-variables
 '(zoom-size 'size-callback)
 '(zoom-ignored-buffer-name-regexps '("^*ediff.*"))')

#+end_src

(defmacro pretty-magit (WORD ICON PROPS &optional NO-PROMPT?)

(defun add-magit-faces ()
  "Add face properties and compose symbols for buffer from pretty-magit."
  (interactive)
  (with-silent-modifications
    (--each pretty-magit-alist
      (-let (((rgx icon props) it))
        (save-excursion
          (goto-char (point-min))
          (while (search-forward-regexp rgx nil t)
            (compose-region
             (match-beginning 1) (match-end 1) icon)
            (when props
              (add-face-text-property
               (match-beginning 1) (match-end 1) props))))))))

(advice-add 'magit-status :after 'add-magit-faces)
(advice-add 'magit-refresh-buffer :after 'add-magit-faces)

    (setq use-magit-commit-prompt-p nil)
